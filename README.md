主要主要用于本地邮件分类处理，以及个人开发环境便捷设置。

## 二、使用方法

### 2.1、环境准备

```
git clone https://github.com/guojd/mailWork.git
cd mailWork
```
### 2.2 快速开始

* 首先安装`jdk1.8`
* 然后，指定参数并用几行代码完成整个Text2SQL fine-tune流程：
```shell
    ./run.sh
```
